# TinyRenderer Practice
这是一个从零手写的 C++ 微型软件光栅化器练习。来自https://github.com/ssloy/tinyrenderer?tab=readme-ov-file

## 核心进度
* 剥离商业引擎黑盒，手撸图形渲染底层逻辑。
* 从基础画线算法起步，推演并实现了无除法的 Bresenham 算法，以规避 FPU 浮点运算带来的 CPU 周期消耗。
* 当前进度代码见 `main.cpp`。

## 学习日记
我将我在推导数学公式、分析底层计算机体系架构（如内存管理、寄存器交互）时的真实思考过程，记录在了 [`diary.md`](./diary.md) 中。这是我坚持第一性原理的学习记录。# TinyRenderer Practice
